 
 	
A hyperlocal mobile app that connects people within a 10 km radius to hire verified individuals for small paid tasks like home repairs, errands, or technical help. Users can negotiate prices and pay via UPI or cash. The app also includes a community feature where neighbors can share urgent needs or emergencies and volunteer to help others, such as assisting senior citizens. It aims to build a trusted local support network.

## UI / UX
The NearNeed app utilizes a premium, **District-style** aesthetic for its user interface. All screens use a universal scrollable layout with a sticky frosted-glass bottom navigation. 

### Core Design Principles:
- **Color Palette**: Navy (`#101D33`), Blue (`#1D5EF3`), Red (`#F43F5E`)
- **Typography**: High-contrast, standard `sans-serif` with `android:textStyle="bold"`, `sans-serif-black` for headers, and `sans-serif-medium` for labels.
- **Components**: Pill-shaped inputs (`bg_district_pill_input`), rounded edge-to-edge buttons (`16dp` corner radius), custom `ChipGroup` selectors for preferences, and custom high-fidelity SVG iconography.
- **Shadows**: `6dp` elevation on card-like sections and upload boxes, with ambient/spot shadows to create a floating "pop" effect.
